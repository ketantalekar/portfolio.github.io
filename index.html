Title: Automatic Tire Pressure Inflation System

Description: Project is concerned about and to develop an “automatic tyre pressure inflation system”. As we are aware that by drop of few pressure units in vehicle its results in the reduction in mileage, tyre life, safety and performance. This system can be placed in every in automobile under any operating condition, this will not only maintain the correct tyre pressure but also increase tyre life, mileage and safety so we have fabricated this system to inflate and deflate the tyre automatically by using control units. This system is named automatic because it checks the tyre pressure continuously using built control device and accordingly gives alert signals to the driver about the tyre condition.
